# ShitTD

A simple CLI tool for managing the shit you've got to do.

Current state of the state:

```
$ shittd --help

A manager for your shit to do list

Usage: shittd <COMMAND>

Commands:
  add
  list
  finish
  clean
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
